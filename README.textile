h1. RTubeToMp3

* http://github.com/maykon/RTubeToMp3

h1. DESCRIPTION

Convert videos from Youtube to files Mp3.

h2. Dependencies

* ruby 1.8.6
* mechanize[http://mechanize.rubyforge.org]
* nokogiri[http://nokogiri.rubyforge.org]
* open-uri

h2. Usage

Put this script in your PATH and download from youtube.com.br like this:
<pre><code>ruby RTubeToMp3.rb Link_to_Youtube</code></pre>

You will find the downloaded chapters under $HOME/Music/RTubeToMp3.

For Windows user you will find downloaded chapters under C:\Documents and Settings\[USERNAME]\Documents\RTubeToMp3.
